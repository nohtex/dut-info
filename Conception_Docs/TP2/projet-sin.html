Thomas FLAMENT - Projet SIN

Pour notre projet de fin d'année (STI2D SIN) nous avons choisi de créer un système d'authentification par badge NFC grâce à une Raspberry.


Nous avons alors utilisé HTML, CSS, PHP ainsi qu'une base de données (MySQL) afin de créer un panneau d'administration permettant à l'administrateur d'ajouter et/ou de retirer des utilisateurs et des badges d'accès.


Pour faire communiquer la Raspberry et la base de données nous avons préféré créer une API en PHP permettant à la Raspberry d'effectuer une requête au site et vérifier l'accès du badge. Le langage de programmation utilisé sur la Raspberry était Python et servait à faire communiquer le lecteur de badge NFC de la Raspberry avec l'API.
